### npm install

**_npm install --save:_** 生成环境的依赖包
**_npm install --save-dev:_** 开发环境依赖包

当在 windows 中通过调用路径去调用 webpack 时，必须使用反斜线();

webpack 中的名词解析：

我们可以通过配置方式指定 loader
**_规则(loader rules)_**、
**_插件(plugins)_**、
**_解析选项(resolve options)_** ，
以及许多其他增强功能。
